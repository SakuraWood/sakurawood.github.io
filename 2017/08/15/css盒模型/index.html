如果你之前接触过开发，有着UI编写经验，这也许是最容易理解的教程。就算之前没接触过开发，也能迅速理解css的盒模型原理。

编写界面，无非是把你想放的元素放到该放的位置上去。

css中的一个基本概念是盒模型，可见元素会在页面中占据一个矩形区域，该区域就是元素的盒子(box)。
